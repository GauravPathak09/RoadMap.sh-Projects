Testimonial Cards Project
A clean, responsive, and visually appealing Testimonial Cards component built using modern CSS techniques. This project focuses on layout precision, hover effects, and a "mobile-first" design approach to showcase user feedback effectively.

🚀 Features
Responsive Grid Layout: Seamlessly transitions from single-column on mobile to multi-column layouts on desktop using CSS Flexbox/Grid.

Modern UI/UX: Features subtle box-shadows, rounded corners, and elegant typography for a professional look.

Hover Animations: Interactive scaling or shadow depth effects to improve user engagement.

Optimized Assets: Uses lightweight profile avatars and scalable icons.

Dark Mode Support: Includes a media query-based dark theme for better accessibility.

🛠️ Tech Stack
HTML5: Semantic structure for better SEO and accessibility.

CSS3: Custom properties (CSS variables), Flexbox, and Media Queries.

Google Fonts: Integrated 'Inter' or 'Poppins' for a clean aesthetic
